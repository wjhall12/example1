What is Emulation


Emulation is when software is being used on non-native hardware, like emulating Android OS through your computer. The emulator is able to act similar to the original hardwear and is why many use it to emulate older consoles to play the games. Emulation is mostly used for video games because many of the old consoles like Nintendo Entertainment System or Playstation 1 will never be released again. There are emulators that do other stuff like bluestacks which is an android emulator for windows; it allows you to play mobile games on that aren't on the system.

What are Roms


Rom stands for read-only memory and in the case of video game roms it is the file containing the data from, a read-only memory chip. The emulator needs many things to make the game run; you need the game in the form of a room and also the bios files which stands for basic input output system. The bios file is used for the emulator to pretend like you are on the actual hardwear and make the experinece better. Both roms and bios files can be dumped from the game and the system itself (like burning a cd) but if you don't have the game on console it will be hard to get it. This is where many people go on the internet and just download the rom of the bios file from a website, and you might think this is illegal, but it isn't.
